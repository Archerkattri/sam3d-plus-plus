# sam3d++

HiCache++ — DMD/Prony EXPONENTIAL velocity forecasting (new method) for SAM 3D Objects' PyTree velocities (slat flow-matching), with HiCache (Hermite) as the comparison baseline. The exponential basis is exact on the feature-ODE solution class where the polynomial drifts, holding quality at larger skip intervals. See `hicache-plus-plus`.
